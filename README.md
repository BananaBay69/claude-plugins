# bananabay-plugins

Claude Code plugin marketplace maintained by BananaBay69.

## Usage

```
/plugin marketplace add BananaBay69/claude-plugins
/plugin install <plugin-name>@bananabay-plugins
```

## Plugins

| Name | Description |
|---|---|
| [bananabay-watchdog](https://github.com/BananaBay69/claude-code-watchdog-plugin) | Emits heartbeat signals for claude-code-watchdog |
